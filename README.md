# Image Processing

Description. 
The package Image-Processing is used to:
		processing:
					- Histrogram matching
					- Structural similarity
					- Resize image
		Ultils:
					- Read image
					- Save image
					- Plot image
					- Plot results
					- Plot histograms

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install image-processing

```bash
pip install image-Processing
```
